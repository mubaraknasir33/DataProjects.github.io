## Featured Projects

### Bike Purchase Behaviour Analysis

![Bike Purchase Dashboard](images/bike-purchase-dashboard.png)

This Excel project analyses customer demographic and lifestyle characteristics to identify patterns associated with bike purchasing.

The analysis explores:

- Customer age brackets
- Average income
- Gender
- Marital status
- Commute distance
- Education
- Regional purchasing behaviour

The project was completed using Excel Tables, PivotTables, PivotCharts, calculated fields, slicers and Report Connections.
